# NetworkX - Content Creator & Sponsor Networking Platform

A modern web application that connects content creators with potential sponsors, facilitating meaningful partnerships and collaborations.

## Features

- **Creator Profiles**: Content creators can create detailed profiles showcasing their niche, audience size, and content style
- **Sponsor Profiles**: Businesses can create profiles with their requirements and campaign budgets
- **Discovery System**: Browse and match with potential partners based on mutual interests
- **Responsive Design**: Fully responsive interface that works on all devices

## Tech Stack

- Frontend:
  - React 18
  - TypeScript
  - Material-UI (MUI)
  - React Router
  - Vite

## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm (v6 or higher)

### Installation

1. Clone the repository
```bash
git clone [your-repository-url]
cd networkx
```

2. Install dependencies
```bash
npm install
```

3. Start the development server
```bash
npm run dev
```

4. Open [http://localhost:5173](http://localhost:5173) to view it in your browser

## Development

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run preview` - Preview production build
